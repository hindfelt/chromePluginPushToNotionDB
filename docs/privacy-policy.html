Privacy Policy for Save to Notion with AI

Last Updated: October 7, 2025

1. Introduction

This Chrome Extension ("Save to Notion with AI") allows you to save web pages to your Notion database with AI-generated summaries and tags. We are committed to protecting your privacy and being transparent about how your data is handled.

2. Data We Collect

The extension collects and processes the following information:

  • Web Page Data: URLs, titles, and content of web pages you choose to save
  • API Credentials: Your Notion API key, Notion Database ID, and Google AI API key (stored locally in your browser)
  • Generated Content: AI-generated summaries, tags, and "why it matters" sections

3. How We Use Your Data

Your data is used for the following purposes:

  • AI Processing: Web page content is sent to Google AI (Gemini) API to generate summaries, tags, and relevance descriptions
  • Notion Integration: Processed data is sent to your personal Notion database via the Notion API
  • Local Storage: API keys and configuration settings are stored locally in your browser using Chrome's storage API

4. Data Storage and Security

  • Local Storage: All API keys and configuration are stored locally in your browser and are never transmitted to our servers
  • No Server Storage: We do not operate any servers that store your data
  • Your Control: All saved data resides in your personal Notion database, which you fully control

5. Third-Party Services

This extension uses the following third-party services:

Google AI (Gemini) API

  • Purpose: Generates AI-powered summaries and tags for web pages
  • Data Shared: Web page URLs, titles, and content
  • Privacy Policy: Google Privacy Policy

Notion API

  • Purpose: Saves processed data to your Notion database
  • Data Shared: Web page URLs, titles, AI-generated summaries, tags, and metadata
  • Privacy Policy: Notion Privacy Policy

6. Data Sharing

  • We do not sell, rent, or share your personal data with any third parties beyond the services listed above
  • Data is only shared with Google AI and Notion as necessary to provide the extension's functionality
  • You control what data is sent by choosing which web pages to save

7. Your Rights and Control

You have complete control over your data:

  • Data Access: All saved data is in your Notion database, accessible at any time
  • Data Deletion: You can delete any saved pages directly from your Notion database
  • Revoke Access: You can uninstall the extension at any time, which will remove all locally stored settings
  • API Key Management: You can revoke API keys through the respective platforms (Google AI, Notion)

8. Children's Privacy

This extension is not intended for use by children under the age of 13. We do not knowingly collect personal information from children under 13.

9. Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated privacy policy.

10. Data Retention

  • API keys are stored locally until you uninstall the extension or clear your browser data
  • Saved web pages remain in your Notion database until you delete them
  • We do not retain any copies of your data on our servers
